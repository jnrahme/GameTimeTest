# Code of Conduct

This repository is a professional take-home project. Contributions, review
comments, issues, and automation output should stay clear, respectful, and
focused on improving the software.

## Expected Behavior

- Use welcoming, precise language.
- Give feedback with concrete examples and suggested fixes.
- Respect different levels of context and avoid personal attacks.
- Keep discussions tied to product quality, maintainability, accessibility, and
  correctness.

## Unacceptable Behavior

- Harassment, insults, threats, or discriminatory language.
- Publicly sharing private information without explicit permission.
- Repeated off-topic comments that disrupt review or issue discussion.
- Misrepresenting test results, security status, or production readiness.

## Enforcement

The repository owner may remove comments, close issues, block contributors, or
decline contributions that violate this code. Serious or repeated violations can
result in permanent removal from repository participation.
